外部連携のためにAPIキーや機密情報を秘匿して管理することができます。
このプラグインを利用することで、機密情報をkintone上に保存し、 kintone.plugin.app.proxy() を利用することで機密情報を隠蔽しながら外部連携を行うことができます。

使い方

以下の手順に従ってください:

  1. 外部連携に利用するための機密情報を作成、取得します。
  2. 取得した機密情報をこの画面から保存します。
    1. URL: 送信先に指定したいURLを指定します。
    2. Method: 通信に利用したいMethodを指定します。
    3. Header: 通信時にリクエストヘッダーに指定したい情報をJSON形式で指定します。
    4. Data: 通信時にリクエストボディに指定したい情報をJSON形式で指定します。
  3. JavaScriptカスタマイズ時に外部連携を kintone.plugin.app.proxy() を利用して行うようにし、プラグインIDは下記に表示されているこのプラグインのIDを指定してください。
